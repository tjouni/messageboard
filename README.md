# messageboard

Keskustelufoorumi, jossa on käyttäjiä. Jokainen käyttäjä voi kuulua useampaan kategoriaan. Viestiketjut näkyvät käyttäjille, jos käyttäjä ja ketju kuuluvat samaan kategoriaan. Yksi ketju kuuluu aina vain yhteen kategoriaan. Käyttäjät voivat kirjoittaa viestejä ketjuihin, joihin heillä on oikeudet. Ylläpitäjä voi luoda foorumille käyttäjiä ja lisätä heille kategorioita/rooleja sekä tarvittaessa poistaa viestejä/viestiketjuja.
